# 📊 SQL Data Warehouse Analytics Project

This project demonstrates advanced data analysis using SQL on top of a dimensional data warehouse built in SQL Server.

The goal of this project is to perform Exploratory Data Analysis (EDA), generate business insights, and practice advanced SQL techniques using fact and dimension tables.

This project follows the data warehouse created in the previous project and focuses on analytics queries used by data analysts and business intelligence teams.


------------------------------------------------------------

## 🚀 Project Objectives

- Explore database structure and metadata
- Analyze dimension tables
- Identify date ranges and historical coverage
- Calculate business metrics
- Perform ranking and cumulative analysis
- Track changes over time
- Perform segmentation analysis
- Generate business reports


------------------------------------------------------------

## 🏗️ Database Used

This project uses the Gold Layer of the data warehouse:

- gold.dim_customers
- gold.dim_products
- gold.fact_sales
